# Vaux LightTable

A local darkroom/explorer for the Vaux 1906 material.

Run standalone:

```bash
cd /mnt/c/Users/chris/BASECAMP/presentation/HermesPresentation/LightTable
./run.sh
```

Open http://127.0.0.1:8766/index.html

If the parent HermesPresentation server is already running on port 8765, open:

http://127.0.0.1:8765/LightTable/index.html

Modes: `1` Light, `2` Sequence, `3` Map guesses, `4` Absence. Hover the main image for the loupe; click to zoom.
